## 健康检查
```bash
curl -sS http://127.0.0.1:8080/v3/health
curl -sS https://api.xgvst.com/v3/health
```

## 断连演练
```bash
launchctl unload ~/Library/LaunchAgents/com.xgvst.api.cloudflared.plist
curl --http1.1 -i https://api.xgvst.com/v3/health
launchctl load ~/Library/LaunchAgents/com.xgvst.api.cloudflared.plist
```
