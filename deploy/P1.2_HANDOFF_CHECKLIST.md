# P1.2 交接清单（用于打通 CF Pages 实际关联）

当前已完成代码与流水线准备，差两项外部接入：

## A. GitHub 远程仓库
在 workspace 根目录执行（或把仓库 URL 发我）：

```bash
cd /Users/mac/.openclaw/workspace
git remote add origin <你的-github-repo-url>
git push -u origin main
git push -u origin develop
```

## B. Cloudflare 鉴权（任选一种）

方式1：在终端完成交互登录
```bash
npx wrangler login
```

方式2：提供 API Token（适合非交互执行）
```bash
export CLOUDFLARE_API_TOKEN=<token>
export CLOUDFLARE_ACCOUNT_ID=<account-id>
```

鉴权完成后我将继续自动执行：
1. 创建/确认 Pages 项目 `xgvst-web`
2. 以 `main` 生产、`develop` 预览触发部署
3. 回传 `*.pages.dev` 公网预览 URL
4. 立刻执行 TTFB + 压缩审计（Sub-C）
