# DoD1 h3 会话复用复核（2026-02-24 14:39）

## 目的
验证“同一浏览器会话下连续请求会从 h2 升级到 h3”的假设。

## 方法
- 通过 Browser Relay 控制同一 Chrome 标签页（`https://xgvst.com/market`）
- 在页面内连续发起 12 次请求：`/api/universe?probe_reuse={i}`
- 使用 Resource Timing `nextHopProtocol` 读取每次请求协议

## 结果
- Navigation: `h2`
- 12次请求协议：
  - `h2`: 11
  - `unknown`: 1（首条采样）
  - `h3`: 0

样本：
- i=1..11 全部 `h2`

## 结论
在同一会话连续请求场景下，当前链路仍稳定为 `h2`，未观察到“自动升级 h3”的证据。
因此 DoD1 仍未达到 PASS 条件，需继续走 Cloudflare 日志口径（http_version）与边缘配置排障。
